  • Library Box

    Library Box is a project started by Jason Griffey, the head of Library IT at University of Tennessee. The tool is a modified MR 20 TP link router. It provides it own wireless signal and connects to any device that has wifi. It becomes sort of a hotspot. Library Box can provide access to the internet in areas where there is no accessability to the web. Best way for students to learn this software is to buy the router and go along with tutorial videos.

  • Zotero

    Zotero is a personal research assistant that automatically senses content in your web browser, allowing you to add it to your personal library. Whether you're searching for a preprint on arXiv.org, a journal article from JSTOR, a news story from the New York Times, or a book from your university library catalog, Zotero has you covered with support for thousands of sites.

  • Libki

    Libki is a web-based open source management tool that allows users to create a computer reservation and time management system. The reservation system can be customized to the user's needs, such as requiring reservations or allowing patrons to immediately use copen computers. One feature is that Libki can manage the amount of time that patrons can be logged on to the library computers. Libki is an easy software to learn and use. Students can look at tutorial videos or the link to useful tips to understand how it works.

  • Omeka

    Omeka is a free, open source, web publishing platform and content managing system that was created for the display of library, museum, archives, and scholarly collections and exhibitions. One feature is that you are able to collaborate with others and make content. Anyone can use the software for many different purposes. The software is also free. There are many tutorial videos to help students get to learn more about the software and it's purpose. The link to useful tips will also help students to get started.

  • Open Exhibits

    Open Exhibits is an initiative that looks to transform the way in which museums and other informal learning institutions produce and share computer-based exhibits. Developers and non-developers alike can use pre-programmed templates to create amazing exhibits.

  • Alice

    Alice is an innovative and open source tool where users can program 3D animations, interactive narratives, or build simple games. It uses a block-based programming environment and teaches logical and computational thinking skills and fundamental principles of coding. A cool feature is that you are able to build your own 3D stories while learning programming. This program is easy to use and takes time to for students to grasp. Students can watch tutorial videos and also experiment with it.

  • Calibre

    Calibre is a cross-platform open-source suite of e-book software. Calibre supports organizing existing e-books into virtual libraries, displaying, editing, creating and conversion of e-books, as well as syncing e-books with a variety of e-readers.

  • Evergreen

    Evergreen is an open source library software. It helps library patrons find library materials and helps libraries to manage, catalog, and circulate the materials. It is used by more than 2,000 libraries around the world. Their database feature looks easy to use and very beneficial for librarians wanting to put information for books. Students wanting to use this software should probably watch tutorials because it could be confusing.

    Wordseer is a text analysis environment that combines visualization, information retrieval, sensemaking and natural language processing to make the contents of text navigable, accessible, and useful.

  • Collective Access

    Collective Access is an open access relational database used for collections management of archives, mueseum collections and special collections. The software is free to use, modify, and share. Some features include the ability to complie printable reports, maps, and diagrams and managing tools for controlling the flow of checked in and out materials.
